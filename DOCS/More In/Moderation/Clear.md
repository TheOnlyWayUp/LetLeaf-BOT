## ***Clear***

> **Keys**
>> 1. **[XYZ]** = *Required*
>> 2. **(XYZ)** = *Optional*

* **Aliases**: `Clear`,`Purge`,`Delete`

* **Usage**: ```>>Clear [NUMBER]```

* **Returns**: *Deletes (Purges) the Number of messages specified. If the Number provided is greater than the total Number of messages in the channel, All the Messages get Deleted!!*

* **Danger Level**: ❌❌❌❌ _
